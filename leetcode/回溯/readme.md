# 回溯算法 backtracking
- 纯暴力搜索法
- 看作是一棵树，深度递归，宽度用for循环，if为终止条件
## 解决的问题
- 组合问题  没有顺序
    - 39
- 切割问题

- 子集问题

- 排列问题  强调元素顺序

- 棋盘问题
    - N皇后
    - 数独问题

## 算法模板
```js
function backtracking(问题描述) {
    if (终止条件) {
        存放结果;
        return ;

    }
    for(选择：本层集合中元素(树中结点孩子的数量就是集合的大小)){
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果JavaScript的React框架;
    }
}
```
